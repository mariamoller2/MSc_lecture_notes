# Your Turn! Exercise: Experimenting with Docker Engine Swarm Mode and Terraform

<img src="https://media.giphy.com/media/13GIgrGdslD9oQ/giphy.gif" width=50%/>


The purpose of this exercise is that you experiment with Docker Engine Swarm Mode and Terraform.

## _ITU-MiniTwit_ in Local Docker Engine Swarm Mode

To experiment with Docker Engine Swarm Mode, navigate to [this repository](https://github.com/itu-devops/itu-minitwit-docker-swarm/tree/local_swarm).
Follow the tutorial in the [`README.md` file](https://github.com/itu-devops/itu-minitwit-docker-swarm/blob/local_swarm/README.md).

Note, make sure to look at the exercise in branch [`local_swarm`](https://github.com/itu-devops/itu-minitwit-docker-swarm/tree/local_swarm).
The main branch of that repository contains an example of instantiating a Docker Swarm cluster on DigitalOcean via DigitalOcean's web-API.

Alternatively, in case you do not want to experiment with our local Docker Engine Swarm Mode setup, work on a Docker Swarm as in the [official tutorial](https://docs.docker.com/engine/swarm/swarm-tutorial).
However, that tutorial requires that you have three Linux (virtual) machines readily available.
