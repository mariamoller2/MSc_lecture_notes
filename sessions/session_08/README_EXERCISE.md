## Exemplary Logging Setup for ITU_MiniTwit

Deploy and investigate our version of [_ITU-MiniTwit_ with logging](https://github.com/itu-devops/itu-minitwit-logging/).
Note, make sure that you can work either on the branch `grafana_loki` of the Git repository at https://github.com/itu-devops/itu-minitwit-logging or on the branch  `master` which contains a similar logging setup but with the ELFK stack.
